# P1.JS-Jogo-da-Velha
Primeiro projeto em JavaScript - Jogo da Velha

Com o Jogo da Velha online é possível:
1. Jogar com 2 jogadores
2. Jogar com IA

O jogo exibe um menu onde o jogador decide com quem jogar.
